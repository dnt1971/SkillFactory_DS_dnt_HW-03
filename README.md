### Описание проекта  
У Василия, основателя компании «Газ-Таз-Ваз-Нефть», дела идут в гору: у него уже функционирует 200 скважин для добычи газа. В этом году он открывает 30 новых скважин. Однако в целях оптимизации расходов и повышения дохода Василию необходимо оценить, сколько денег будет приносить ему каждая из скважин, а также понять, какие факторы (параметры скважин) потенциально сильнее всего повлияют на объём добычи газа. Для этого Василий решил нанять вас как специалиста в области Data Science.

:arrow_up:[к оглавлению](#оглавление)

### Какой кейс решаем?    
Нужно построить регрессионную модель, которая прогнозирует выработку газа на скважине (целевой признак — Prod) на основе остальных характеристик скважины, и проинтерпретировать результаты вашей модели.

**Что практикуем**     
Учимся 
  - Знакомимся с моделями линейной регрессии с применением методов линейной алгебры.
  - Правильно настраивать модели и делать подбор гиперпараметров.
  
:arrow_up:[к оглавлению](#оглавление)


### Краткая информация о данных
Вам представлены данные об оттоке клиентов некоторого банка. Файл с исходными данными вы можете скачать [здесь](https://github.com/dnt1971/SkillFactory_DS_dnt_HW-03/blob/master/data/unconv.csv).

**Признаки в данных:**

* Well — идентификатор скважины;
* Por — пористость скважины (%);
* Perm — проницаемость скважины;
* AI — акустический импеданс ($кг/м^2 * 10^6$);
* Brittle — коэффициент хрупкости скважины (%);
* TOC — общий органический углерод (%);
* VR — коэффициент отражения витринита (%);
* Prod — добыча газа в сутки (млн. кубических футов).

:arrow_up:[к оглавлению](#оглавление)

### Этапы работы над проектом  
1. Разведывательный анализ данных.
2. Построение модели линейной регрессии методом матричного МНК и анализ моледи методами линейной алгебры.
3. Сравнение модели с библиотечсным аналогов в sklearn.
4. Построение модели линейной регрессии с полиномиальными коэффициентами 3го порядка
   - по умолчанию
   - с регуляризацией Lasso с подбором гиперпараметров
   - с регуляризацией Ridge с подбором гиперпараметров
   - с регуляризацией ElasticNet с подбором гиперпараметров
5. Сравнение всех моделей.
6. Выгрузка в GitHub
7. Документирование проекта

Файл программы с результами работы [можно скачать тут](https://github.com/dnt1971/SkillFactory_DS_dnt_HW-03/blob/master/MATH_ML_2_Практика.ipynb).
